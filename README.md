# "Puissance 4" like game

## Prerequisites

Install `Python 3 (>= 3.6.4)`
Install a venv if needed

## Run the game

### Install dependancies

```bash
    ./venv/bin/pip3 install -r requirements.txt
``` 

### Run it

```bash
    ./venv/bin/python3 main.py
```
